This allows the user access to past, present data and as data is generated over time the user will receive access to the reports going forwards from when the pharmacy was created on ISP and also up to the date that the pharmacy started trading i.e. if a pharmacy started trading from June 2023, got access to ISP in Sept 2023, they would see the information dated back to June 2023.
