COVID-19 vaccination programme: 1 April 2024 to 31 August 2025 - Site selection process for suppliers About your proposed service - IT user 1

New designated sites need to be set up on the IT systems that facilitate participation in the COVID-19 vaccination service as part of the necessary onboarding process. We need the contact information of two IT Users (with @nhs.net email addresses) to allow for vaccine supply and other accounts to be created. IT Users will be set up on systems such as:

  • the Foundry system (accessed to review information held about the site, reporting and vaccine ordering)
  • the NHS Future Collaboration platform (accessed to review guidance and other relevant information about the COVID-19 vaccination programme)
  • National Booking Service (NBS) administrators (accessed to set up other NBS users and schedule appointments)
  • preferred point of care system and administrators if relevant (accessed to record vaccinations)

If you are a site with existing IT Users no changes will be made to these as a result of the information provided here. If you want to change these please follow the process set out in the IT System Access & Use - COVID-19 Vaccination Programme - FutureNHS Collaboration Platform

This must be an @nhs.net username of an individual (and not generic / shared email accounts). For ‘Other Providers’, if you do not have an @nhs.net email address, please leave this blank and you will be assigned one during the onboarding process.
